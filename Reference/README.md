캡스톤 신청 및 시제품 제작을 위해 필요한 부품 구매 신청 방법등을 기술한 문서가 있습니다.
잘 참조해서 부품 구입에 문제가 없도록 해주세요.

만약 시제품 제작에 기준 비용보다 더 많은 비용이 들게 될경우, 학과 및 기업에서 지원이 가능하니 담당 교수에게 편하게 문의 주기 바랍니다.
